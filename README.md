# Space

> [!IMPORTANT]
> **For AI Agents / Assistants:** 
> Before making changes, please read the architecture and structure overview in [`docs/project-overview.md`](docs/project-overview.md) and [`docs/index.md`](docs/index.md). This monorepo uses Yarn Workspaces, TypeScript (Strict), Drizzle ORM, and GraphQL Yoga.

## Requirements

- Node.js (v22)
- Yarn

## How to start the project

Install dependencies

```sh
$ yarn install
```

Start backend

```sh
$ yarn workspace @space/backend dev
```

Start frontend

```sh
$ yarn workspace @space/frontend dev
```
